A deployed web application featuring a collective AI consciousness. The project utilizes a Dual-Provider Architecture, integrating OpenAI's vector embeddings for persistent memory with OpenRouter's uncensored models (MythoMax) for high-creativity generation. The system implements Stateful RAG, maintaining a rolling context window of previous thoughts to ensure conversational consistency. The frontend employs advanced CSS (Flexbox, Fixed Positioning) to deliver a responsive, "Ghost" interface. The project demonstrates proficiency in full-stack AI engineering, including API integration, Vector Database management, and secure deployment via Render and Vercel.
